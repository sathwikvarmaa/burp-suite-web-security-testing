# Burp Suite Web Security Testing

This repository documents my practical work using Burp Suite for web application security testing.

## Tools Used

- Burp Suite
- Kali Linux
- Firefox

## Topics Covered

- HTTP request interception
- Request manipulation
- Burp Repeater testing
- Burp Intruder payload testing
- Input validation testing
- Web application traffic analysis

## Purpose

The goal of this repository is to document hands-on testing performed while learning web application security using Burp Suite.
