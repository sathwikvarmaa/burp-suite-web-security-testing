# Testing Requests using Burp Repeater

## Objective
Modify and resend HTTP requests to analyze application responses.

## Steps

1. Intercepted request using Burp Proxy.
2. Sent request to Repeater.
3. Modified request parameters.
4. Sent request multiple times.

## Example

Original request

username=admin&password=12345

Modified request

username=admin&password=test
